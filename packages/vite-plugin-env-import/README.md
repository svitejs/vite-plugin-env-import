# @svitejs/vite-plugin-env-import

PROJECT TAGLINE HERE

## Installation

```bash
pnpm i -D @svitejs/vite-plugin-env-import
```

## Usage

### add to vite config

```ts
// vite config
import { defineConfig } from 'vite';
import { envImport } from '@svitejs/vite-plugin-env-import';

export default defineConfig({
	plugins: [
		envImport({
			/* plugin options */
		})
	]
});
```
